I accept that this tool is not a substitute for clinical judgement.

After you have entered your values select an iron preparation below

  • First line for inpatients is Cosmofer®
  • If Cosmofer® is contraindicated, or the patient is fluid overloaded, select Monofer®
  • Ferinject® can be used if Monofer® is not suitable, for day case patients, or if Cosmofer® would delay a discharge
